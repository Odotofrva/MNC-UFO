AR-UFO Transmission

AR-UFO Desktop + Mobile Viewer

This version is built to make the model load more reliably, show clearer status messages, and expose a visible AR launch button on supported phones. On Android Chrome, real-world AR needs a secure page served from HTTPS or localhost, not a file opened directly from your downloads folder.

What to expect

  • Desktop should show the 3D UFO in the viewer.
  • Supported mobile devices should show a View in your space button.
  • After AR starts, move the phone slowly so the device can detect a flat surface.

Launch requirements

  • Use Chrome on Android for the best ARCore flow.
  • Serve this page by HTTPS or localhost.
  • Keep retro_ufo.glb in the same folder as this HTML file.
